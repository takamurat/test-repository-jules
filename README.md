# オセロゲーム

シンプルなウェブベースのオセロ（リバーシ）ゲームです。

## 特徴

- 8x8のインタラクティブなゲームボード
- オセロの完全なゲームロジック（有効手の配置、石の反転、パス、ゲーム終了判定）
- 黒と白のスコア表示
- 現在のターン表示
- ゲームをリセットする機能

## 遊び方

1. `index.html` ファイルをウェブブラウザで開きます。
2. ゲームが開始されます。あなたのターン（最初は黒）になったら、石を置きたい有効なマスをクリックします。
3. ゲームが終了すると、勝者がアラートで表示されます。
4. 「リセット」ボタンを押すと、いつでも新しいゲームを開始できます。

## ファイル構成

- `index.html`: ゲームの基本的なHTML構造です。
- `style.css`: ゲームボードや石の見た目を定義するスタイルシートです。
- `script.js`: ゲームのすべてのロジック（ルール、操作など）を記述したJavaScriptファイルです。
